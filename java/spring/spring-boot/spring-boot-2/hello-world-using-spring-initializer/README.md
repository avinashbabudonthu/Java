# Spring Boot 2 project using spring initializer

## Steps
* Open link [Spring Initializer](http://start.spring.io)
* Select Project, Language, Spring Boot version, Dependencies
* Click on button **Generate the Project**
* Zip fill will be downloaded
* Extract and import to IDE
