# Columnar database
* Column oriented
* Optimized for reading and writing columns of data rather than rows of data
* Good for things like Data warehouse and analytics
* Reduces disk io requirements
* Examples
* [Apache cassandra](../README.md#cassandra)
* Apache HBase
* Amazon Redshift
