# DB Queries
* Create database
```
create database [if not exists] database-name;
create schema [if not exists] database-name;

create database if not exists practice;
```
* Show databases
```
show databases;
```
* Use database
```
use database_name
```
* Drop database
```
drop database database-name;
```
