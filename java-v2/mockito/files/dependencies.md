# Mockito dependency
* Add below dependency in pom.xml. Check central repository for latest version
```
<dependency>
	<groupId>org.mockito</groupId>
	<artifactId>mockito-core</artifactId>
	<version>2.1.0-RC.2</version>
	<scope>test</scope>
</dependency>
```
